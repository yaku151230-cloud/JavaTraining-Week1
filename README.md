実行方法
1.このプログラムを "Janken.java" という名前で保存
2.ターミナルを開く
3.ファイルを保存したフォルダに移動
4.コンパイルをする（ `javac Janken.java` を入力）
5.ターミナルに `java Janken` を入力してゲームを実行する
6.表示された指示に従い、0, 1, 2 のいずれかを入力
7.勝敗が判定される


工夫した点・学んだこと
・visual studio codeを使用している中で、Run Codeを押して実行すると、文字化けが起きてしまう可能性があること（Run Javaを使用することで、UTF-8を自動的に変更して実行してくれること）
・Javaの初歩的な操作（最初に入力しないといけないテンプレートが存在しているなど）
・visual studio codeで入力していると、AIが入力するであろう内容を先に提示してくるが、実際にその内容が正しい時が非常に多いこと
